# IAM Implementation in AWS

## Steps

1. Logged into AWS Management Console
2. Created IAM User
3. Assigned permissions (AmazonS3ReadOnlyAccess)
4. Created User Group
5. Added user to the group
6. Created IAM Role (EC2)
7. Tested login using IAM user

---

## Output

* IAM user created successfully
* Permissions assigned to user
* Group created and user added
* Role created successfully
* Access denied for restricted services (verifies IAM working)
